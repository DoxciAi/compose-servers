# Requirements
1. This was tested with Docker version `23.0.1` and docker-compose version `1.25.0`.
2. The `.env` file is expected in the same directory as `docker-compose.yaml`
# Run
In terminal:
```
docker-compose up
```

# Access GUI/Swagger
Visit `127.0.0.0/docs` in browser.

# Available Services
- `API`: port 80
- `redis`: port 6379
- `rabbitmq`: port 5672
